# Iris Dataset Logistic Regression

This repository contains two versions of logistic regression analysis on the Iris dataset:

## Folders
- `with_ydata_profiling/`: Includes exploratory data analysis using ydata-profiling.
  - `iris_dataset(with ydata_profiling).ipynb`: Jupyter notebook
  - `Iris_dataset_EDA.html`: EDA report

- `without_ydata_profiling/`: Basic analysis without ydata-profiling.
  - `iris_dataset(without ydata_profiling).ipynb`: Jupyter notebook

## Requirements
- Python 3.x
- Libraries: pandas, scikit-learn, ydata-profiling (for with version), numpy, matplotlib, seaborn

## Usage
Navigate to the desired folder and run the notebook.
